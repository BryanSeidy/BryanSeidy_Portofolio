# Images Directory

Ce dossier contient toutes les images utilisées dans le portfolio.

## Structure

### `/images/skills/`
Placez ici les icônes des compétences techniques. Les fichiers doivent être nommés comme suit :
- `html-css.png`
- `javascript.png`
- `typescript.png`
- `react.png`
- `nextjs.png`
- `nodejs.png`
- `python.png`
- `php.png`
- `mongodb.png`
- `mysql.png`
- `git.png`
- `wordpress.png`

**Recommandation** : Format PNG ou SVG, taille recommandée 64x64 pixels minimum.

### `/images/projects/`
Placez ici les images de présentation des projets. Les fichiers doivent être nommés comme suit :
- `civika.png`
- `cinema-scraper.png`
- `sessrail.png`
- `patrie-art.png`
- `builder-brothers.png`
- `art-sous-le-manguier.png`

**Recommandation** : Format PNG ou JPG, taille recommandée 800x600 pixels minimum, ratio 4:3 ou 16:9.

## Où trouver des icônes ?

Pour les icônes de compétences techniques, vous pouvez utiliser :
- [Simple Icons](https://simpleicons.org/)
- [DevIcon](https://devicon.dev/)
- [Icons8](https://icons8.com/)
- [Flaticon](https://www.flaticon.com/)
